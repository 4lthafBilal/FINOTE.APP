## 1. Deskripsi
FINOTE merupakan aplikasi web yang memuat pencatatan pemasukan & pengeluaran, mengelola keuangan, dan memantau transaksi yang anda lakukan. Aplikasi ini bertujuan untuk membantu pengguna dalam pengelolaan keuangan pribadi melalui antarmuka yang sederhana dan fitur pencatatan yang efisien

## 2. User Story
Sebagai user, saya ingin:
- Mendaftarkan akun agar data keuangan saya aman dan personal.
- Mencatat pemasukan dan pengeluaran agar dapat terorganisir.
- Melihat total saldo secara real-time di halaman utama.
- Melihat riwayat transaksi terbaru untuk memantau aktivitas keuangan harian.

## 3. Software Requirements Specification (SRS)
Feature List:
- Sistem Autentikasi: Fitur Sign Up dan Sign In bagi pengguna.
- Dashboard Overview: Menampilkan total saldo, total pemasukan, dan total pengeluaran.
- Manajemen Transaksi: Input data transaksi dengan kategori pendapatan atau pengeluaran.
- Manajemen Profil: Mengelola informasi pribadi pengguna.
- Sidebar Navigasi: Akses cepat ke Home, Dompet, Transaksi, Analisis, dan Profil.
